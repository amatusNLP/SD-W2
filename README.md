to use the Pretrained Model launch sse.jar via command line
general commands:
- load a model: "load MODEL_NAME"
- get similarities between two words: "gs WORD1 WORD2 [MEASURE]"
- get neighbors of a word: "gn WORD [NUMBER][MEASURE]"
